# Path Of Crafting

간단한 웹 프로토타입으로, 게임의 핵심 흐름을 3구역(Inv / Pro / Stat) 레이아웃으로 확인할 수 있습니다.

## 실행 방법
브라우저에서 `index.html` 파일을 직접 열면 됩니다. 별도의 빌드나 서버 설정이 필요 없습니다.

## 화면 구성
- **Inv (1)**: 인벤토리. 재료/장비 보유량을 표시합니다.
- **Pro (2)**: 제작 공간. 재료 조합과 결과 아이템을 확인할 수 있습니다.
- **Stat (3)**: 캐릭터 능력치. 제작 후 상승한 값을 보여줍니다.

메인 영역은 1:2:1 비율로 배치되어 있어 첫 화면에서 게임 흐름을 직관적으로 파악할 수 있습니다.
