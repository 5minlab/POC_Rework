화면 구성

  • 1번 구역: Inv (가방/재료)

배치 비율 1 : 2 : 1

Inv

Pro

Production

재료를 조합해 아이템을 제작하는 구역입니다.

버튼을 눌러 예시 조합을 확인하세요.

Stat

게임 흐름: 재료 확인 → 조합 → 결과 확인 → 능력치 상승
